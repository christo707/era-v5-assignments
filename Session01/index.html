Part 1 · S1-1

Activations exist for a reason

A model with no nonlinearity can only draw a straight boundary, so it cannot separate two interleaved / concentric rings. Adding a single ReLU hidden layer can. Only the activation changes — the boundary picture is the money shot.

The claim

A purely linear model (a single layer + sigmoid) can only carve space with one straight line. Concentric rings aren't linearly separable, so it's stuck near chance.

The concept

Stacking linear layers is still linear. A nonlinearity like ReLU bends space, letting the network compose many lines into a curved, closed boundary that wraps the inner ring.

What we built

~300 noisy 2D points as two rings (inner = class 0, outer = class 1). Both models train live in your browser; we plot each decision boundary and its accuracy.

What to watch: the linear model can only tilt one straight line — it splits the rings roughly in half and parks near ~55%. The ReLU model bends the boundary into a closed loop around the inner ring and climbs to ~99–100%. Same data, same training; the only difference is one nonlinearity. Try dropping hidden units to 1 — the curve almost disappears and accuracy collapses back toward linear.
